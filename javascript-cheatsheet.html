Introduction to JavaScript

JavaScript is a programming language that, together with HTML and CSS, is one of the core technologies of the World Wide Web. When it comes to web development, JavaScript is one of the most popular languages owing to its features and capabilities.

1. JavaScript Fundamentals

  • Writing JavaScript code in an HTML page: We must enclose the Javascript code within the <script> tag in order to include it on an HTML page just as the example shown below

    • <script type = "text/javascript">

    //JavaScript coding can be done inse this tag

    <script>

    With this information, the browser can correctly make out the code is written in JavaScript and execute the code

  • Inclusion of external JavaScript files in an HTML file: An external JavaScript file can also be written separately and included within our HTML file. This gives better organization.

    • <script src= "script.js> <script>

  • Usage of Comments in JavaScript coding:Comments are extremely useful inf= programming. It helps others understand your code better.There are two ways
    • Single-line comments: If you want to include a single line comment, start it with "//"
    • Multi-line comments: If you want to write a comment that spans multiple lines, you can wrap it in /* and */ to prevent it from being executed.
